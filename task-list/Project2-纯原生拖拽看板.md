# Project 2: 纯原生拖拽看板 (Kanban Drag & Drop)

> 来源：[模型能力测试题目.md](../模型能力测试题目.md#-project-2-纯原生拖拽看板-kanban-drag--drop)

## 🎯 考察核心

- 原生 HTML5 Drag and Drop API 深入理解
- DOM 状态与数据模型的同步
- 复杂事件代理与边界处理

## 📝 任务描述

请不使用任何第三方库，纯手写一个**类 Trello 的任务看板面板**。

包含"待办"、"进行中"、"已完成"三列。用户可以自由地将任务卡片在不同列之间拖拽，也可以在同一列中拖拽排序。拖拽经过其他卡片时，要有视觉反馈（如挤开一个空位）。数据需实时保存到 `localStorage` 中，刷新页面状态不丢失。
