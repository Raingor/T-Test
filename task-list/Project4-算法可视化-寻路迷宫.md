# Project 4: 算法可视化 - 寻路迷宫 (Pathfinding Visualizer)

> 来源：[模型能力测试题目.md](../模型能力测试题目.md#-project-4-算法可视化---寻路迷宫-pathfinding-visualizer)

## 🎯 考察核心

- 浏览器事件循环 (Event Loop) 的非阻塞渲染
- 图论算法（A\* 或 Dijkstra）
- 网格系统的 CSS Grid 布局

## 📝 任务描述

构建一个 **A\* 寻路算法的可视化工具**。

页面渲染一个 30x30 的网格。用户可以点击网格设置"起点"、"终点"，按住鼠标拖拽可以"画出不可通过的墙壁"。点击"开始寻路"后，要求以动画的形式，**一步一步（以肉眼可见的速度）展示算法搜索的蔓延过程**，搜索过的节点变成蓝色，最终找到的路径变成黄色。
