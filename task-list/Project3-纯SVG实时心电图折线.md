# Project 3: 纯 SVG 实时心电图/行情折线 (Real-time SVG Ticker)

> 来源：[模型能力测试题目.md](../模型能力测试题目.md#-project-3-纯-svg-实时心电图行情折线-real-time-svg-ticker)

## 🎯 考察核心

- SVG 路径 (`<path d="...">`) 的数学构建
- 实时数据流的视图映射
- 内存管理与垃圾回收

## 📝 任务描述

请不依赖 ECharts 等任何图表库，用纯 SVG 和 Vanilla JS 实现一个**"实时股票行情折线图"**。

通过 `setInterval` 模拟每 100 毫秒推送一个新数据点。图表需要像心电图一样，始终只显示最近 50 个数据点，线条平滑，并且随着新数据的到来，折线要平滑地向左滚动。图表的 Y 轴极值需要根据当前 50 个点的最大值/最小值**自动缩放（Auto-scaling）**。
