# Project 1: Canvas 物理与引力模拟器 (Particle Physics)

> 来源：[模型能力测试题目.md](../模型能力测试题目.md#-project-1-canvas-物理与引力模拟器-particle-physics)

## 🎯 考察核心

- HTML5 Canvas 高性能渲染机制
- 基础运动学与碰撞检测算法
- 空间坐标系的映射

## 📝 任务描述

请使用原生 HTML5 Canvas 构建一个"引力弹球模拟器"。

页面中心有一个隐藏的"引力源"。在页面上随机生成 500 个颜色各异的小球。小球需要受到中心引力的吸引向中间靠拢，同时小球之间具备物理碰撞体积（撞击后发生弹性反射）。要求有流畅的动画（60fps），并且当浏览器窗口大小改变时，Canvas 画面不能被拉伸变形。
