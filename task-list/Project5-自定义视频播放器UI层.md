# Project 5: 自定义视频播放器 UI 层 (Custom Video Player)

> 来源：[模型能力测试题目.md](../模型能力测试题目.md#-project-5-自定义视频播放器-ui-层-custom-video-player)

## 🎯 考察核心

- HTML5 `<video>` API 深入交互
- 复杂鼠标事件的相对坐标计算
- 状态防抖与时间同步

## 📝 任务描述

隐藏 HTML5 原生的视频控件，从零开始手写一套**自定义视频控制栏**。

必须包含：播放/暂停按钮、音量滑动条、时间显示、全屏切换，以及一个**可点击、可拖拽的进度条**。当用户拖拽进度条时，视频画面需要实时预览拖拽位置的帧。
